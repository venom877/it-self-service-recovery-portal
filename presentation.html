              1. Startup Trigger (Automatic logon execution)
            

            

              For bank employees, waiting to discover a technical failure mid-transaction or on an active WFH call is disastrous. Our software automates diagnostics immediately when the system boots or as soon as a user logs in. This proactively cleans up storage bottlenecks, adjusts MTU limits, and clears locked Citrix caches before the employee even opens their first application.
            

            

              Startup Configuration Method:
              Registry Run Key: HKLM:\Software\Microsoft\Windows\CurrentVersion\Run
            

          


          
          

            

              
Secure Packaging & Intune

              

                Scripts are packaged inside a silent Microsoft Installer (MSI) or executable utilizing PSAppDeployToolkit, allowing 1-click deployment through standard Microsoft Intune to 100% of corporate remote PCs.
              

            

            

              
Group Policy Enforced PKI

              

                Group policies restrict execution to "AllSigned". All diagnostic modules are code-signed with the bank's Enterprise root certificate. If malicious actors modify the script, the signature breaks and Windows blocks execution.
